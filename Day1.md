# Day 1 — 2026-06-22

## Topic: Kickoff & Monorepo Walkthrough

Today was our first day as a team. We sat together and talked about the whole project. I saw how the folders are set up: one folder for the frontend (React), one for the backend (Express), and one for Python scripts. All of this lives inside one single Git repository, which is called a monorepo. We also decided how we will name our branches, like feature/dilpreet-something, so it is easy to know who is working on what.

**What I learned today:** A monorepo keeps everything in one place, so it is easier to share code and run the whole app together.
